从github克隆项目在本地运行

从github克隆项目(react,vue)在本地运行

1.克隆项目

git clone '地址'

2.首先安装nodejs最新稳定版，配置环境变量，使得node和npm命令能正常运行

3.项目下安装依赖

npm install

这个命令会自动分析package.json配置进行安装的

4.运行项目

npm start
