我用react的感想是：这才是一个生态体系。抛开全家桶以及衍生出来的一堆旁支，就UI库来说就有material-ui,AntDesign这两大库来支持react，真心赞。

2.新建项目，进入

$ create-react-app antd-demo
$ cd antd-demo

３．安装并引入antd

$ npm i antd --save
